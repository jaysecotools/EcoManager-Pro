Step 2: Import into EcoManager Pro

           

After downloading, follow these steps to import the data:

           
                   
  1. Open the EcoManager Pro tool
    2.                
  2. Navigate to the "Data" section using the sidebar
    3.                
  3. Click the "Import Data" button
    4.                
  4. Select the JSON file you downloaded
    5.                
  5. Choose whether to overwrite existing data or merge with current data
    6.                
  6. Click "Import Data"
